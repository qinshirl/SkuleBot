$$| x | = { \left\{ \begin{array} { l l } { - x , } & { \quad x < 0 } \\ { x , } & { \quad x \geq 0 } \end{array} \right. } \quad 6 - | 2 x | = { \left\{ \begin{array} { l l } { 6 + 2 x , } & { \quad x < 0 } \\ { 6 - 2 x , } & { \quad x \geq 0 } \end{array} \right. }$$

4. Find $\lim_{x \to 2}(x-2)\cos \left(\frac{1}{(x-2)^{2}}\right)$.

AB6 CS2

This definitely requires the Squeeze Theorem, with positive and negative cases.

Case I: x - 2 > 0, or x > 2.

$$\begin{array} { r } { - 1 \leq  \cos \left( \frac { 1 } { ( x - 2 ) ^ { 2 } } \right) \leq  1 } \\ { - ( x - 2 ) \leq  ( x - 2 ) \cos \left( \frac { 1 } { ( x - 2 ) ^ { 2 } } \right) \leq  ( x - 2 ) } \end{array}$$

$$\lim_{x\to2}-(x-2)=\lim_{x\to2}(x-2)=0$$

Therefore, by the Squeeze Theorem, the limit in case I goes to zero.

Case II is nearly identical and we can get away with saying “Case II is done similarly.” Here it is in full:

Case II: $x-2<0$, or $x<2$.

$$-1 \leq  \cos \left(\frac{1}{(x-2)^{2}}\right) \leq  1$$

$$- ( x - 2 ) \geq ( x - 2 ) \cos \left( { \frac { 1 } { ( x - 2 ) ^ { 2 } } } \right) \geq ( x - 2 )$$

$$\lim_{x\to2}-(x-2)=\lim_{x\to2}(x-2)=0$$

Therefore, by the Squeeze Theorem, the limit in case II goes to zero.

Therefore, the limit is zero.
