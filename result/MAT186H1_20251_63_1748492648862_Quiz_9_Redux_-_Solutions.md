4. Find $\lim_{\theta \to 0^{+}}(\sin \theta)^{\tan \theta}$.

This limit is of the form $0^{0}$, so L'Hopital's rule applies.

$$\begin{array} { r l } & { L = \operatorname* { l i m } _ { \theta \to 0 ^ { + } } ( \sin \theta ) ^ { \tan \theta } } \\ & { \ln L = \operatorname* { l i m } _ { \theta \to 0 ^ { + } } \tan \theta \ln ( \sin \theta ) } \\ & { = \operatorname* { l i m } _ { \theta \to 0 ^ { + } } \frac { \ln ( \sin \theta ) } { \left( \frac { 1 } { \tan \theta } \right) } } \\ & { \stackrel { L \prime H } { \cong } \operatorname* { l i m } _ { \theta \to 0 ^ { + } } \frac { \left( \frac { 1 } { \sin \theta } \cdot  \cos \theta \right) } { \left( -  c s c  ^ { 2 } \theta \right) } } \\ & { = \operatorname* { l i m } _ { \theta \to 0 ^ { + } } - \sin \theta \cdot  \cos \theta = 0 } \\ & { \therefore L = e ^ { 0 } = 1 } \end{array}$$
